О чём этот сайт?

Этот сайт посвящен игре в стиле песочница. Тебе нужно из шарниров, конечностей и суставов собрать робота. В будущем его можно будет программировать.

Как в это играть?

Просто перетащи сустав с панели инструментов на игровое поле чтобы создать новый сустав. Перетащи один сустав к другому чтобы соединить их костью. Такими простыми манипуляции собери робота. Нажми кнопку старт и посмотри как его размажет по земле. Увы гравитация при полном отсутствии силы трения пока работает только так.
